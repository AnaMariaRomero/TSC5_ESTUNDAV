live_help ¿Cómo calcular las cerámicas que necesito para cubrir la habitación?

Tenemos que calcular cuántos metros cuadrados tenemos en la habitación, para ello debemos calcular el área de dicha habitación. La habitación donde tenemos que colocar las cerámicas es rectangular de 3 metros (m) de altura por 6 metros (m) de base por lo que el área se calcula multiplicando base x altura (A=bxh)
